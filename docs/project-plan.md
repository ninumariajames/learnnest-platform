# LearnNest

## Vision
Online learning platform for Pre-KG to Grade 12 students.

## Student Features
- Register
- Login
- Watch Videos
- Download Notes
- Take Quizzes

## Admin Features
- Upload Videos
- Upload PDFs
- Manage Courses
- Manage Students

## Cloud Stack
- AWS EC2
- AWS RDS
- AWS S3
- Docker
- NGINX
- GitHub Actions
- CloudWatch
